# Museo Virtual - Refranes Argentinos

Un sitio web interactivo que celebra la cultura argentina a través de sus refranes.

## Características
- Galería de arte temática.
- Sala de refranes con narración de audio.
- Información sobre artistas destacados.
- Diseño responsivo y moderno.

## Tecnologías Utilizadas
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Cómo Ver el Proyecto
1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador para ver la página principal.
